Bookmarks

ℹ️ Storage Types

  • Local Storage: Faster, stored only on this device
  • Sync Storage: Slower, but syncs across all your Chrome browsers (max 100KB)

⌨️ Klavye Kısayolları

  • Alt + B - Görünür mesajı bookmark'la/çıkar
  • Alt + Shift + B - Bookmark panelini aç/kapat

ℹ️ Nasıl Çalışır?

  • Mesajların üzerine geldiğinizde bookmark butonu görünür
  • Bookmark'lanmış mesajlar panelde ve sidebar'da listelenir
  • Bookmark'a tıklayarak o mesaja hızlıca gidebilirsiniz
  • Export/Import ile bookmark'larınızı yedekleyebilirsiniz

💾 Yedekleme

Export/Import: Bookmark'larınızı JSON dosyası olarak yedekleyebilir ve geri yükleyebilirsiniz. Chrome sıfırlandığında bookmark'larınızı kaybetmezsiniz!

Emoji Markers

⭐ Favori Emojiler

İşaretleme yaparken hızlı erişim için favori emojilerinizi seçin. Sürükle-bırak ile sıralayın.

ℹ️ Storage Types

  • Local Storage: Faster, stored only on this device
  • Sync Storage: Slower, but syncs across all your Chrome browsers (max 100KB)

ℹ️ Nasıl Çalışır?

  • Mesajların üzerine geldiğinizde marker butonu görünür
  • Favori emojileriniz hızlı seçim için gösterilir
  • "..." butonuna tıklayarak tüm emojilere erişebilirsiniz
  • Marker'lanmış mesajlar badge ile işaretlenir
  • Sabit buton (📍) ile tüm marker'ları görüntüleyin
  • Export/Import ile marker'larınızı yedekleyebilirsiniz

💾 Yedekleme

Export/Import: Emoji marker'larınızı JSON dosyası olarak yedekleyebilir ve geri yükleyebilirsiniz.

Sidebar Collapse

ℹ️ Nasıl Çalışır?

  • Sidebar'daki Starred ve Recents section'larına chevron (▶/▼) ikonu ekler
  • Section başlığına veya chevron'a tıklayarak listeyi açıp-kapatabilirsiniz
  • Collapsed durumda sadece başlık görünür, tüm liste gizlenir
  • Çok fazla yıldızlı mesaj veya recent conversation olduğunda sidebar daha düzenli görünür
  • "Durumu Hatırla" seçeneği ile tercihleriniz kaydedilir

💡 İpucu

Bu özellik sidebar'ı daha temiz ve organize tutar. İhtiyacınız olmayan section'ları kapatarak sadece ilgilendiğiniz alana odaklanabilirsiniz!

Content Folding

💻 Code Block Folding

💬 Message Folding

⚙️ General

ℹ️ Nasıl Çalışır?

  • Message Folding: Hem user hem Claude mesajlarına chevron (▶/▼) ikonu ekler (sol üst köşe)
  • Mesajları katlamak için chevron'a tıklayın - ilk N satır + "Show full message" görünür
  • Heading Folding: Claude response'lardaki başlıklara (h1-h6) chevron (▶/▼) ikonu ekler
  • Başlığa tıklayarak altındaki içeriği gizleyip gösterebilirsiniz (Obsidian tarzı)
  • Hiyerarşik folding: Üst seviye başlık kapatılınca alt başlıklar da kapanır
  • Code Block Folding: Uzun kod bloklarına collapse butonu ekler (sağ üstte)
  • Collapsed durumda ilk birkaç satır + "Show X more lines" butonu görünür
  • Chevronlar ve butonlar sadece hover'da görünür (collapsed olmadıkça)
  • State'ler conversation bazında hatırlanır (opsiyonel)

✨ Obsidian/VSCode Tarzı Folding

Bu özellik Obsidian ve VSCode'daki fold/unfold sistemini Claude'a getiriyor. Uzun response'ları organize etmek ve sadece ilgilendiğiniz bölüme odaklanmak için mükemmel!
